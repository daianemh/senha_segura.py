# 🔐 Verificador de Senhas Seguras

🔍 Sobre o projeto

O Verificador de Senhas Seguras é um projeto simples desenvolvido em Python, com o objetivo de aplicar conceitos básicos de cibersegurança e boas práticas de segurança digital.

Ele analisa senhas digitadas pelo usuário e classifica sua força como Fraca, Média ou Forte, além de exibir dicas de como criar senhas mais seguras.

Este projeto faz parte dos meus estudos na área de Cibersegurança e Programação Back-end (Java/Python), integrando aprendizado prático com segurança da informação.

⚙️ Funcionalidades

✅ Verifica a força da senha digitada
✅ Analisa critérios importantes de segurança:

Comprimento mínimo (8 caracteres)

Letras maiúsculas e minúsculas

Presença de números

Inclusão de caracteres especiais (@ $ % & * ! ?)
✅ Classificação automática da senha (Fraca / Média / Forte)
✅ Exibe recomendações de melhoria

💻 Tecnologias utilizadas

Linguagem: Python 3

Biblioteca: re (expressões regulares nativas do Python)

Terminal interativo: entrada e saída de texto no console
